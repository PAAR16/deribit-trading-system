# Deribit Trading System

A C++ based trading system for Deribit exchange using WebSocket API.

## Building

1. Ensure CMake is installed.
2. Create build directory: `mkdir build && cd build`
3. Configure: `cmake ..`
4. Build: `cmake --build .`

## Usage

Run the executable: `./trading_system`

## Dependencies

- CMake 3.10+
- C++17 compiler
- (Optional) websocketpp, Boost for WebSocket support
